Личный ассистент
===============================================


Данный проект создается с целью отработки навыков создания микросервисов с использование различных технологий и инструментов.

Планируемые модули:
- Личный дневник
- Список задач
- Заметки
- Финансовый помошник
- Напоминания 
- Уведомления
- Избранные новости

------

## **Личный дневник** ##

Стэк:

- Python

- Django ORM
- DB (Вид БД в процессе выбора)
